## 📝 نظرة عامة (Overview)
هذا المشروع عبارة عن صفحة هبوط (Landing Page) مصممة بمنتهى الاحترافية باستخدام **HTML5** و **Tailwind CSS**. الهدف من الصفحة هو التسويق وعرض **4 نماذج (Demos)** لقوالب مواقع شخصية (سيرة ذاتية، معرض أعمال، مستقل، وبطاقة ذكية) مع توجيه العميل للتواصل مباشرة عبر الواتساب لطلب تنفيذ موقعه.

---

## ✨ المميزات الرئيسية (Features)
- 📱 **تجاوب كامل (100% Responsive):** تصميم ملائم تماماً لجميع الشاشات (موبايل، تابلت، ديسكتوب) مع قفل التمدد الجانبي (Horizontal Scroll) على الموبايل نهائياً.
- 🎨 **تصميم حديث (Modern UI):** استخدام تأثيرات (Hover/Shadows) عصرية لرفع مستوى تجربة المستخدم (UX).
- 💻 **واجهة تخيلية (CSS Mockup):** تم استبدال الفيديوهات الثقيلة بواجهة متصفح تخيلية مبنية بالكامل باستخدام Tailwind CSS لزيادة سرعة الصفحة وإعطاء طابع برمجي احترافي.
- 🚀 **تأثيرات حركية (AOS Animations):** دمج مكتبة AOS لدخول العناصر بسلاسة عند التمرير (Scroll).
- 💬 **أزرار تواصل ذكية (Smart CTAs):** روابط الواتساب مجهزة مسبقاً برسائل نصية (`أهلاً بشمهندس مينا، أريد تصميم...`) لتسريع عملية المبيعات.

---

## 🛠️ التقنيات المستخدمة (Technologies)
- **HTML5**: لبناء هيكل الصفحة.
- **Tailwind CSS (CDN)**: لتنسيق الصفحة بالكامل وبسرعة فائقة.
- **FontAwesome (v6.4.0)**: لجميع الأيقونات المستخدمة في الموقع.
- **AOS Library**: لعمل الأنيميشن (Fade, Zoom).
- **Google Fonts (Cairo)**: الخط العربي الأساسي للموقع.
